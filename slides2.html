Commands & Concepts 🛠️

Your Claude toolkit

Tokens How Claude reads — every word, emoji & space costs tokens (its "fuel")
CLAUDE.md A file that gives Claude permanent context about your project
/model Switch between models — Sonnet (fast) vs Opus (smartest)
/clear Reset the conversation when Claude gets confused or off track
Plan mode Ask Claude to plan before coding — think first, build second (shift+tab)
MCP Plug-ins that give Claude superpowers — browse the web, search docs, call APIs & more
Skills Reusable prompt templates — save a good prompt once, use it anytime with /skill

Prompting Tips 💡

How to talk to Claude and get great results

🎯
Be Specific
"Add a red 30px bird on the left" beats "add a bird"
🧱
Break It Down
Don't ask for the whole game at once — start small, build up
🔄
Iterate
Not perfect? Say "make it bigger" or "change the color" — keep going
🐛
Debug Together
Something broke? Tell Claude what happened vs. what you expected
📋
Give Context
Tell Claude what exists, what you want, and why — context = better answers
